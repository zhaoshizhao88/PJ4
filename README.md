# 🧠 PJ4：基于可视化的计算机专业考研院校推荐系统

## 项目简介

PJ4 是一个结合数据分析与可视化技术的院校推荐系统，旨在为准备考研的计算机专业学生提供个性化、数据驱动的院校选择建议。项目通过爬虫收集高校信息，结合用户输入的偏好与成绩，生成推荐列表，并以图形化方式展示结果，提升决策效率。

## 📦 项目结构

```
PJ4/
├── .idea/                         # IDE 配置文件
├── python75f6hd0t/                # Python 核心逻辑模块
├── spider/                        # 网络爬虫模块，用于抓取高校数据
├── 基于可视化的...数据库文档.doc    # 数据库设计与说明
```

## 🔧 技术栈

- **前端**：HTML, Vue.js, JavaScript
- **后端**：Python
- **数据处理**：Pandas, SQLite
- **可视化**：ECharts / Matplotlib（视具体实现而定）

## 🚀 功能亮点

- 🎯 **个性化推荐**：根据用户输入的目标地区、成绩、偏好等参数，智能匹配适合的院校。
- 📊 **数据可视化**：推荐结果以图表形式展示，直观易懂。
- 🕸️ **自动爬虫**：实时抓取高校官网及第三方平台数据，保持信息更新。
- 🗂️ **数据库支持**：结构化存储高校信息，支持快速查询与筛选。

## 📄 使用说明

1. 克隆项目：
   ```bash
   git clone https://github.com/zhaoshizhao88/PJ4.git
   ```

2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

3. 运行爬虫收集数据：
   ```bash
   python spider/spider_main.py
   ```

4. 启动推荐系统：
   ```bash
   python python75f6hd0t/main.py
   ```

5. 打开前端页面查看推荐结果
